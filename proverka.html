Шаг 1 из 5 · Согласие

Согласие на проверку

Верификация проводится только с вашего согласия.

Как это работает. Мы подтверждаем ваш опыт по официальной выписке из электронной трудовой книжки (СТД-СФР с Госуслуг) и диплом по реестру ФИС ФРДО. Итог — сертификат «Проверено», который вы прикладываете к резюме.

Личность

Загрузите разворот паспорта с фото. Файл остаётся в браузере.

Сейчас — визуальная сверка ФИО с выпиской. Фаза 2: подтверждение личности через вход по Госуслугам (ЕСИА) — государственная идентификация.

Опыт работы — СТД-СФР

Загрузите выписку из электронной трудовой книжки (файл XML с Госуслуг). Платформа прочитает места работы автоматически.

Где взять: Госуслуги → Справки → «Выписка из электронной трудовой книжки» → скачать в формате XML. Данные формирует Социальный фонд России.

Образование

Данные диплома для сверки в госреестре ФИС ФРДО.

Диплом сверяется по госреестру ФИС ФРДО (Рособрнадзор) по этим данным. Проверка выполняется вручную оператором: у реестра нет открытого API, а поиск защищён капчей — автоматически из браузера это сделать нельзя. В сертификат образование попадает только после сверки.

Проверьте сводку

Так будет выглядеть основа вашего сертификата. Отправьте на выдачу — мы завершим сверку и пришлём результат.

Честно: сертификат подтверждает то, что реально проверено. Опыт вне официальной трудовой (фриланс, ГПХ, самозанятость, зарубеж) в СТД-СФР не отражается и помечается отдельно.

Проверка обработана
